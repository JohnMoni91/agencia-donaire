# 📸 Agência Donaire - Model Agency Management System

A robust web platform designed to streamline operations for model agencies, featuring profile management, scheduling, and document verification flows.

## 🚀 Project Overview
Developed to solve the logistical challenges of managing talent, this application handles the entire lifecycle of a model's registration — from initial sign-up and document verification to casting scheduling.

## 🛠️ Tech Stack
- **Core:** PHP 8, Laravel
- **Database:** MySQL
- **Frontend:** Blade, Bootstrap, JavaScript
- **Architecture:** MVC Pattern

## ✨ Key Features
- **Profile Management (CRUD):** Complete lifecycle management for model profiles.
- **Document Verification:** Secure handling of contracts and ID verification flows.
- **Approval Workflow:** Admin interface to review and approve/reject new applications.
- **Scheduling System:** Integrated calendar for managing bookings and castings.

## 💡 Dev Note
This project utilizes modern development practices. AI tools were used to accelerate boilerplate generation, with manual implementation and review of core business logic and security features.

## 🔒 Privacy Notice
Sensitive data (real contracts and user PII) has been excluded from this repository in compliance with data protection regulations.

# 📸 Agência Donaire - Sistema de Gestão

Sistema web completo para gerenciamento de uma agência de modelos, focado na administração de perfis, agendamentos e segurança de dados.

## 🚀 Sobre o Projeto
Este projeto foi desenvolvido como uma solução fullstack para centralizar as operações de uma agência. O sistema permite o cadastro de modelos, validação de documentos legais (contratos) e gestão de agenda.

## 🛠️ Tecnologias Utilizadas
- **Back-end:** PHP 8, Laravel Framework (MVC)
- **Front-end:** Blade Templates, Bootstrap, JavaScript
- **Banco de Dados:** MySQL
- **Outros:** Git, Composer

## ✨ Funcionalidades Principais
- **Gestão de Perfis:** Criação, edição e exclusão de perfis de modelos.
- **Fluxo de Aprovação:** Sistema de verificação onde administradores aprovam ou rejeitam novos cadastros.
- **Verificação Documental:** Upload e validação de documentos e contratos (compliance).
- **Agendamento Inteligente:** Calendário integrado para casting e sessões de fotos.
- **Controle de Acesso:** Níveis de permissão diferentes para Modelos e Administradores.

## ⚠️ Nota sobre Privacidade (LGPD)
Este repositório contém apenas o código-fonte. Por motivos de segurança e privacidade (LGPD), todos os arquivos de contratos reais e dados de clientes foram removidos ou ofuscados.
